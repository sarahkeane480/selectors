This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

Here's what it looks like:

Inside my-div
class

classes can be applied to more than one item on a page

Here's how we hook a class attribute to an element:

class="my-list"

Here's how the CSS is applied to the element:

li.my-list{
    background-color:orange;
}

Here's what it looks like:

  • I'm in a li with the class my-list
  • I am not in a list... just kidding, i was li-ing :)
  • I am in an ordered list... just kidding, i was li-ing :)
compound

compound selectors target multiple selectors, which are seperated by commas.

This lets us apply the same rules to several selectors.

div#new-div, li.new-list{
    background-color:blue;
}

Here's what it looks like:

I'm inside a div, and I have an id of new-div
  • I'm a li, and I have a class called new-list
  • I'm styled alongside new-div with a compound selector!
child

A child selector allows us to access the next immediate elements inside the current element.

This lets us apply the same rules to several selectors.

ul.my-kids > li {
    background-color:cyan;
}

Here's what it looks like:

  • I'm in an li inside a ul with a class of my-kids
